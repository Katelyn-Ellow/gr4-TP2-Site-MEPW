# TP2 du cours de « 1W1 Mise en page Web »
## Le bar à chats
### Description
Les bars à chats sont devenus des lieux de détente uniques où les visiteurs peuvent savourer une boisson chaude en compagnie de félins adorables. Inspiré par des établissements comme Café Chat l'Heureux à Montréal, notre espace vise à offrir une ambiance chaleureuse axée sur le bien-être animal et la connexion humaine.
